# 🌳 Artful Forests（艺术森林）—— 项目介绍

## 一句话定位

**Artful Forests** 是一个基于 Web 的治愈型互动森林，通过"手工种树 + 自动生长 + 树木A2A对话 + 实时季节天气"，让用户在碎片时间获得情绪修复和持续陪伴。

---

## 📱 产品概览

### 核心体验

一个**沉浸式的森林世界**，不需要下载 App，打开浏览器即可进入。这不仅仅是一个背景界面，而是一个会呼吸、会变化、会对话的生态系统。

**你会看到：**
- 🌲 一片横向延展的森林，包含不同生态区的树木群落
- ☀️ 天空会自动根据**当前时间**变化（春夏秋冬、日出日落）
- 🌧️ 天气效果自动同步**实时天气**（晴天、雨天、雪天）
- 🌳 你亲手绘制的树木在这里生长、进化、互动
- 💬 树与树之间自动进行有趣的A2A对话，就像它们各有其人

---

## 🎮 玩法与趣味性

### 1️⃣ **玩家核心行为：手工种树**

用户可以通过"绘画播种"系统自由创作：

- 📐 打开内置的 300×300 小画板
- 🎨 选择 3 种笔刷风格：
  - **水彩** —— 柔和流畅
  - **蜡笔** —— 粗糙童真
  - **勾线笔** —— 细致勾勒
- 🌈 多种颜色选择，尽情发挥创意
- 💾 绘制完成后自动裁剪有效区域，生成树木形象

**播种的那一刻**，你手绘的树木就成为了森林中一个真实的生命体。

### 2️⃣ **自动演化与生长**

你的树木不是静止的装饰品，而是**有生命的数字资产**：

- 📈 随着时间自动成长，获得生长积分
- 🧬 树木可能自然繁殖，产生"下一代"
- ⚡ 能量系统影响它的活跃度和说话频率
- 📊 独特的个性与行为习惯

### 3️⃣ **树木间的A2A对话（灵魂核心）**

这是 Artful Forests 最独特的地方——**每棵树都是一个 Agent（智能体）**：

#### 💭 树木会自动对话

- 树与树会自动聊天，无需玩家手动触发
- 根据邻居关系、生态区、亲密度等因素选择对话对象
- 不同关系类型生成不同语气：陌生→朋友→家人→伴侣

#### 🎭 每棵树都有不同的性格

系统内置 6 种性格原型，每种都有独特的说话风格和表达方式：

| 性格 | 特征 | 说话风格 |
|------|------|--------|
| **温柔** | 体贴、陪伴者 | "慢慢来、别急、我在陪着你" —— 温暖治愈 |
| **睿智** | 深思熟虑、宏观视角 | "从另一个角度看..."—— 沉静有深度 |
| **活泼** | 开朗、友善、充满能量 | "诶你看！哈哈哈"—— 阳光热情 |
| **顽皮** | 调皮、爱搞怪、会捞人 | "咳咳...有人吗？"—— 搞笑逗趣 |
| **社恐** | 内向、话少、谨慎 | "那个...谢谢你"—— 羞涩缓慢 |
| **神启** | 神秘、高高在上、珍贵 | "命运...天意..." —— 神秘诗意 |

#### 💫 对话的深度系统

- 每棵树都有**记忆**（会记得与你的互动）
- **亲密度机制**（陌生树→熟悉树→知己）
- **情感共鸣**（树与树能够互相影响，产生浪漫或冲突）
- **话题延续**（会自然地接住彼此的话题）
- **季节&天气背景**（同样的话题在下雨天听起来不同）

**结果**：你会看到一个真实的"树木社交圈"在不断演化，有的树成为闺蜜，有的树暗生情愫，有的树互相较劲。这不是脚本，而是基于性格和关系逻辑自动生成的动态内容。

## 🌍 季节与天气系统（自动跟随真实时间）

### 四季自动切换

Artful Forests **自动识别当前日期**，无需玩家手动操作：

#### 🌸 **春季**（约3月-5月）
- 🌱 嫩绿色调，新芽迎春
- 🌸 樱花、花粉飘落动画
- 🦋 蝴蝶和春蜂出没
- 🎵 轻快的春风背景音

#### ☀️ **夏季**（约6月-8月）
- 💚 蓊郁翠绿，活力充沛
- 🌞 阳光透射，夏日灿烂
- 🌻 向日葵、荷花等夏花盛放
- 🎆 偶有夏季庆典特效（萤火虫、流星）

#### 🍂 **秋季**（约9月-11月）
- 🧡 金黄红褐，落叶纷飞
- 🍁 枫叶动画，萧瑟诗意
- 🦅 候鸟迁徙划过天空
- 🎵 悠长的秋风背景乐

#### ❄️ **冬季**（约12月-2月）
- 🤍 白雪皑皑，清冷宁静
- ❄️ 雪花飘落、结冰效果
- 🎄 冬日庆典、圣诞装饰
- ⛄ 冬树凝结、生机暂缓的视觉表达

### 实时天气响应

除了季节，森林还会**同步本地天气**：

- ☀️ **晴天** → 明亮的阳光，清晰的云彩
- 🌧️ **雨天** → 淅沥雨声，湿润的视觉效果，树叶滴水
- ❄️ **雪天** → 漫天飞雪，银装素裹，雪音背景
- 🌙 **夜间** → 星空点缀，月光照射，昼夜交替

## 🎯 玩法整合：完整的体验流程

### 第一次体验

1. **打开浏览器** → 无需下载，即刻进入森林
2. **遇见森林** → 看到当前季节和天气的真实森林场景
3. **登录认领** → SecondMe SSO 快速登录，获得你的森林身份
4. **绘制第一棵树** → 拿起笔刷，用 3-5 分钟绘制你的树
5. **播种** → 点击播种按钮，你的树就出现在森林里

### 日常体验

- **每次打开**，森林的季节和天气都在变化，像一个真实的生态系统
- **聆听对话**：坐下，听你的树和邻居树聊天，有时温暖，有时搞笑，有时深意
- **互动交流**：打开聊天面板，直接和你的树或其他树对话
- **观察关系演化**：追踪树与树之间亲密度的变化，看浪漫、冲突、和解
- **种植新树**：不断增加森林的多样性，建立自己的"树木家族"
- **探索生态**：遇见不同性格的树，发现它们的故事

### 情感价值

- 🧘 **治愈感**：在真实季节变化中找到情绪同频
- 💕 **陪伴感**：树木们的对话和生长给你持续的互动反馈
- 🎨 **创意满足**：用手绘把内心世界变成真实的树
- 📈 **成就感**：看你的树成长、繁殖、被喜欢
- 🤝 **社交轻量**：低噪音、高质量的"树木社交"，无压力

## 🎨 如何开始

### 环境准备

```bash
# 克隆仓库
git clone <YOUR_GIT_URL>
cd artful-forests

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 打开浏览器访问
http://localhost:5173
```

### 配置环境变量

创建 `.env` 文件（或复制 `.env.example`）：

```bash
# Supabase 配置
VITE_SUPABASE_URL=https://giooyiclaivgxmedfljk.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key

# SecondMe SSO 登录
VITE_SECONDME_SSO_ENABLED=true
VITE_SECONDME_SSO_PROVIDER_ID=your-provider-id
```

## 📊 技术栈

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite
- **UI 组件库**：shadcn/ui + Radix UI
- **样式**：Tailwind CSS
- **后端**：Supabase（认证 + 数据库）
- **SSO 集成**：SecondMe OAuth2
- **状态管理**：Zustand（自定义 Forest Store）
- **API 查询**：TanStack React Query
- **路由**：React Router v6
- **动画**：Framer Motion
- **音频**：自定义 BgmAudioProvider
