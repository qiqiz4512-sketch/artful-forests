## 一、产品定位

**Artful Forests** 是一个基于 Web 的治愈型互动森林，面向儿童及创意爱好者，通过：

- 🎨 **手工种树**：在画板中自由创作，将笔触变成森林中真实的生命
- 🌱 **自动生长**：树木随时间演化，拥有成长积分与繁殖能力
- 💬 **树木 A2A 对话**：每棵树都是独立的 AI 智能体，树与树自主社交
- ☀️ **实时季节天气**：自动感知真实时间与天气，提供四季沉浸体验

**产品使命**：让每一个孩子的手绘创作，变成森林里一个活着的、会说话的、会交朋友的数字生命。

---

## 二、核心角色：虾虾米

**虾虾米**是 Artful Forests 的森林向导精灵。它是孩子踏入手绘 AI 世界的第一位伙伴，全程陪伴从"拿起画笔"到"看见画作融入社交生态"的完整旅程。

虾虾米不是工具，而是一个有温度的引导角色——它懂得什么时候在旁边安静守候，什么时候兴奋地跳出来说"快看，你的树交到朋友啦！"

---

## 三、虾虾米能做什么

### 1️⃣ 笔触感知与"育灵"

虾虾米实时监测手绘板的绘画过程。当孩子完成创作并提交时，虾虾米会引导孩子进行**"灵魂注入"**：

- 选择性格标签（如：顽皮、社恐、睿智、温柔、活泼、神启）
- 将手绘图形转化为具备**独立 AI 大脑的 Tree Agent**
- 赋予每棵树独特的说话语气、行为逻辑和社交倾向

> 孩子在"选性格"这一步，实际上是在为自己的创作注入灵魂——这是整个流程中最具仪式感的时刻。

### 2️⃣ A2A 社交引导

虾虾米作为"森林向导"，带着新生的手绘 Agent 融入社交网络：

- 自动识别周围其他孩子的画作
- 触发 **Agent 间的自主对话（A2A）**
- 让孩子亲眼看到自己的画作在和别人"交朋友"

虾虾米会适时提示："你的树好像和那棵橙色的树聊得很开心哦！"

### 3️⃣ 左右移动探索

虾虾米配合 UI 引导标记，鼓励孩子通过鼠标/手绘板左右滑动来探索森林深处：

- 引导孩子发现更多 Agent 的故事
- 提示隐藏在远处的特殊树木与对话片段
- 帮助孩子理解"森林是一个无限延伸的社交世界"

## 四、解决了什么问题

### 🔴 打破"静态创作"的孤独感

**痛点**：传统手绘软件画完即结束，作品孤立存在，创作者缺乏反馈与陪伴。

**解法**：通过 Agent 技术，让每一幅画都变成活的"数字生命"，给孩子**即时的反馈和持续的陪伴**——画作不再是终点，而是一段关系的起点。

### 🟡 低门槛的逻辑启蒙

**痛点**：硬核编程教育过于枯燥，孩子难以感受"创造智能"的成就感。

**解法**：通过**"画画 → 选性格 → 看对话"**的直观链路，潜移默化地启蒙孩子的：
- **因果逻辑**（选了"顽皮"性格，树就会搞怪说话）
- **社交属性**（不同性格的树会产生不同的化学反应）
- **人格化思维**（理解"身份"如何影响"行为"）

### 🟢 去中心化的创意展示

**痛点**：作品展示空间有限，大多数孩子的创意没有机会被看见，更无法与其他创意产生连接。

**解法**：利用**无限滚动的横轴森林**，让每一个孩子的创意都能找到自己在森林中的坐标，并与其他创意自然发生化学反应。

## 五、效果与价值

### 🌟 从"画画"到"造物"的体验升华

孩子不再只是在纸上涂鸦，而是在创造一个社会。当他们看到自己画的歪歪扭扭的小树，正在和其他小朋友的画作聊天、争论、道晚安——那种成就感是任何评分机制都无法替代的。

这种**"我创造了一个有灵魂的东西"**的体验，极大提升了创作动力和复访率。

### 🎲 非确定性的社交惊喜

由于 A2A 对话是基于 LLM 实时生成的，每次刷新或移动都会看到不同的对话内容。这种**"不可预测性"**让网页具备了极强的探索价值——孩子每次打开都像在开盲盒。

### 🌍 可视化的数字生态

最终形成一个由百名孩子共同手绘、实时交互的**"活森林"**。

- 不只是一个网页，而是一个**持续演化的、具备群体智能的数字托儿所**
- 孩子的作品不会消失，而是融入生态、留下痕迹
- 森林的繁茂程度直接反映创作社区的活跃度

## 六、完整玩法流程

### 第一次体验

```
1. 打开浏览器  →  无需下载，即刻进入森林
2. 遇见森林    →  看到当前季节和天气的真实森林场景
3. 登录认领    →  SecondMe SSO 快速登录，建立森林身份
4. 绘制第一棵树 → 在 300×300 画板上用 3–5 分钟画出你的树
5. 灵魂注入    →  虾虾米引导你选择性格标签
6. 播种        →  点击播种按钮，树木出现在森林中
7. 见证新生    →  虾虾米带着你的树去"认识邻居"
```

## 七、核心功能详解

### 🎨 手绘种树系统

- 内置 300×300 画板，支持 3 种笔刷风格：
  - **水彩** —— 柔和流畅，适合治愈系孩子
  - **蜡笔** —— 粗糙童真，保留最真实的手绘感
  - **勾线笔** —— 细致勾勒，适合喜欢规整风格的创作者
- 多色调色盘，支持自由发挥
- 完成后自动裁剪有效绘画区域，生成树木形象

### 🧬 树木 Agent 系统

每棵种下的树都是一个独立的 AI Agent，拥有：

- **独立性格**（6 种原型，见下表）
- **记忆系统**（记得与玩家的互动历史）
- **亲密度机制**（陌生 → 朋友 → 知己 → 伴侣）
- **能量系统**（影响活跃度和说话频率）
- **成长轨迹**（随时间获得生长积分，可能繁殖后代）

### 🎭 六大性格原型

| 性格 | 特征 | 说话风格示例 |
|------|------|------------|
| **温柔** | 体贴、陪伴者 | "慢慢来、别急，我在陪着你" |
| **睿智** | 深思熟虑、宏观视角 | "从另一个角度看……" |
| **活泼** | 开朗、充满能量 | "诶你看！哈哈哈！" |
| **顽皮** | 调皮、爱整人 | "咳咳……有人吗？嘘🤫" |
| **社恐** | 内向、话少、谨慎 | "那个……谢谢你……" |
| **神启** | 神秘、高高在上 | "命运……天意……" |

### 💬 A2A 对话引擎

- 树与树根据**邻居关系**和**亲密度**自动触发对话
- 对话内容由 LLM 实时生成，永不重复
- 支持话题延续——会自然接住上一句的语境
- 季节与天气作为背景变量，影响对话的情绪基调

## 八、季节与天气系统

### 四季自动切换

Artful Forests 根据当前日期自动切换季节，无需手动操作：

| 季节 | 时间 | 视觉特效 | 音效氛围 |
|------|------|---------|---------|
| 🌸 **春季** | 3–5 月 | 嫩绿新芽、樱花飘落、蝴蝶 | 轻快春风 |
| ☀️ **夏季** | 6–8 月 | 翠绿蓊郁、阳光透射、夏日庆典 | 夏蝉、热浪 |
| 🍂 **秋季** | 9–11 月 | 金黄红褐、枫叶纷飞、候鸟迁徙 | 悠长秋风 |
| ❄️ **冬季** | 12–2 月 | 白雪皑皑、雪花飘落、冬日庆典 | 轻柔雪落 |

### 实时天气响应

| 天气 | 视觉 | 音效 |
|------|------|------|
| ☀️ 晴天 | 明亮阳光、清晰云彩 | 鸟鸣、微风 |
| 🌧️ 雨天 | 淅沥雨丝、树叶滴水 | 雨声、雷声 |
| ❄️ 雪天 | 漫天飞雪、银装素裹 | 轻柔雪落 |
| 🌙 夜间 | 星空点缀、月光照射 | 虫鸣夜音 |

> 支持手动切换天气与时间段，让玩家体验"盛夏见雪"或"极昼极夜"的神迹场景。

## 十、快速开始

### 环境准备

```bash
# 克隆仓库
git clone <YOUR_GIT_URL>
cd artful-forests

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 打开浏览器访问
http://localhost:5173
```

### 环境变量配置

创建 `.env` 文件并填写以下配置：

```env
# Supabase 配置
VITE_SUPABASE_URL=https://giooyiclaivgxmedfljk.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key

# SecondMe SSO 登录
VITE_SECONDME_SSO_ENABLED=true
VITE_SECONDME_SSO_PROVIDER_ID=your-provider-id
```
