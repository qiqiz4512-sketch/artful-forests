# Artful Forests 路演展示计划书

## 1. 路演目标

- 在 7-10 分钟内让评委明确三件事：
  - 这是一个什么产品（治愈型互动森林）
  - 为什么值得做（情绪价值 + 轻社交 + 低门槛 Web 传播）
  - 为什么我们能做成（已有可运行 MVP 与清晰商业化路径）
- 争取达成至少一个结果：
  - 拿到下一轮深度沟通机会
  - 获得种子合作意向（品牌联名/内容共创/校园场景）
  - 获得试投或小额天使关注

## 2. 一句话定位与开场话术

### 2.1 一句话定位
Artful Forests 是一个基于 Web 的治愈型互动森林，通过“种树成长 + 场景生态变化 + 角色对话”，让用户在碎片时间获得情绪修复和持续陪伴。

### 2.2 30 秒开场话术（可直接念）
我们发现，很多人需要放松，但不想再被复杂手游和高噪音社交打扰。Artful Forests 想做一件简单的事：让用户打开浏览器，就能进入一片会随季节、天气、昼夜变化的森林，亲手种下自己的树，并与森林角色互动，形成长期陪伴。我们不是再做一个“打发时间”的产品，而是在做一个“可持续停留”的情绪空间。

## 3. 受众与核心痛点

- 目标用户：18-30 岁学生与年轻职场人，偏好轻疗愈、审美化内容、低门槛体验。
- 核心痛点：
  - 传统游戏流程重、负担高
  - 纯工具型解压产品留存弱
  - 社交产品信息噪音大，缺少安静空间
- 我们的解法：
  - 不下载，网页即开即用
  - 用“生态变化”提供持续新鲜感
  - 用“种植与互动”建立个人情感资产

## 4. 产品亮点（结合当前可演示能力）

- 沉浸生态层：季节、天气、昼夜变化联动视觉氛围。
- 核心行为层：手动种树 + 自动种植，形成个人森林成长轨迹。
- 情感互动层：聊天面板与树木社交关系，强化陪伴感。
- 账号体系层：已接入 Supabase 登录注册（邮箱/用户名 + 密码）。
- 体验强化层：粒子、音效、风铃等多感官反馈，提升“停留时长”。

## 5. 路演结构（10 分钟版本）

### 5.1 时间分配
1. 问题与机会（1 分钟）
2. 产品方案（2 分钟）
3. 现场 Demo（3 分钟）
4. 商业模式（1.5 分钟）
5. 增长与里程碑（1.5 分钟）
6. 融资/合作诉求（1 分钟）

### 5.2 每段要点
1. 问题与机会
- 情绪恢复需求增长，轻量互动产品供给不足。
- Web 形态天然利于传播和冷启动。

2. 产品方案
- 展示三层结构：生态氛围、成长机制、互动关系。
- 强调“低门槛 + 高情绪价值 + 可持续回访”。

3. 现场 Demo
- 登录进入森林
- 种下一棵树
- 切换天气/时段/季节
- 触发聊天互动和生态反馈

4. 商业模式
- Freemium：免费核心体验，订阅解锁深度内容。
- 收入来源：订阅、主题皮肤、限定树种、联名合作。

5. 增长与里程碑
- 冷启动：内容平台短视频 + 社群共建。
- 留存：每日轻任务 + 每周生态事件。
- 里程碑：MVP 打磨 -> Beta 商业化 -> 联名扩展。

6. 融资/合作诉求
- 希望获得产品迭代与增长验证阶段的资源支持。
- 欢迎品牌联名、校园活动、心理健康场景合作。

## 6. 现场 Demo 脚本（3 分钟）

### 6.1 Demo 前准备
- 提前登录测试账号，避免临场网络或验证码风险。
- 浏览器全屏，关闭无关通知。
- 准备 1 个备用演示视频（30-45 秒）防止突发故障。

### 6.2 Demo 讲解词（示例）
1. 进入首页：
现在看到的是我们的森林主场景，它会根据季节和时间变化氛围。

2. 演示种树：
我现在种下一棵树。对用户来说，这不是一次点击，而是“拥有一个会成长的数字生命资产”。

3. 演示生态变化：
切到雨天和夜晚，你会看到粒子和氛围同步变化，这种环境反馈是用户持续停留的关键。

4. 演示互动：
我们打开聊天与社交互动，让森林从“静态背景”变成“有回应的空间”。

5. 收束：
所以我们把放松、陪伴和成长放在同一个产品闭环里，用户每次回来都会有新的感受。

## 8. 评委常见问题与回答模板

1. 你们和普通治愈游戏有什么区别？
我们的差异是“生态场景 + 情感互动 + 可成长资产”三者融合，不只是单点玩法或视觉包装。

2. 为什么从 Web 开始而不是 App？
Web 更适合冷启动和传播验证，降低获客摩擦。验证后再扩展 PWA 或移动壳，能更稳地提升效率。

3. 商业化会不会破坏治愈体验？
我们采用渐进式商业化，免费层保证核心体验完整，付费主要提供个性化与收藏价值，不做强压式付费。

4. 护城河在哪里？
护城河来自持续的内容系统、生态参数调优、用户森林资产沉淀，以及围绕联名和社区形成的品牌心智。

## 10. 7 天冲刺计划（路演前）

1. 第 1-2 天：统一叙事
- 锁定一句话定位、目标用户和核心价值。
- 完成 PPT 初稿与 Demo 路线。

2. 第 3-4 天：强化可信度
- 补齐关键埋点口径和阶段目标。
- 整理可展示的产品截图与操作动线。

3. 第 5 天：压力测试
- 彩排 3 次（计时 + 临场问答）。
- 演练网络异常时的备份播放流程。

4. 第 6-7 天：精修表达
- 精炼开场 30 秒和结尾 20 秒。
- 调整语速、停顿和关键页转场。

Artful Forests 正在把“解压”从一次性消费，变成可持续回访的数字森林体验。我们已经完成核心产品能力，下一步希望与投资方和品牌方一起，把这片森林做成年轻用户愿意长期停留的互动空间。
